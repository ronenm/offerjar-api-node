# offerjar-api
The official node OfferJar API wrapper

## Instalation
```
npm install offerjar-api
```

## Usage
```javascript

OfferJar = require('offerjar-api');

partner = new OfferJar.partner(["partner_token"]);
partner.connect("partner_uid",function(error,response,body) {});

```
